# 隐私权政策 (Privacy Policy)

**生效日期：** 2025年12月12日

感谢您使用 **AI Mock**（以下简称“本扩展”）。本隐私权政策旨在向您说明我们如何处理您的数据。

## 1. 数据收集与使用 (Data Collection and Usage)

**我们不收集任何个人数据。**

- **本地存储**：本扩展的所有核心功能（如 Mock 规则、Mock 响应数据、配置项）均存储在您浏览器的本地存储（`chrome.storage.local`）中。这些数据完全保留在您的设备上，不会上传至我们的服务器或任何第三方服务器。
- **网页内容**：本扩展仅在您浏览网页时拦截网络请求以提供 Mock 功能，不会读取、存储或传输网页内容、Cookie 或任何敏感信息。

## 2. 第三方服务 (Third-Party Services)

- **AI 生成功能**：如果您选择使用本扩展的“AI 生成”功能，您输入的接口描述信息将发送至 DeepSeek API 以生成 Mock 数据。
  - 该过程仅在您主动点击生成按钮时发生。
  - 您的 API Key 仅保存在您浏览器的本地存储（`localStorage`）中，本扩展不会将其上传或同步到其他地方。
  - 除此之外，本扩展不与任何其他第三方服务共享数据。

## 3. 数据安全 (Data Security)

由于本扩展不收集任何数据，因此不存在数据泄露的风险。您的配置数据由 Chrome 浏览器的安全沙箱机制保护。

## 4. 变更 (Changes)

如果我们决定更改隐私政策，我们将在此页面上发布更改。

## 5. 联系我们 (Contact Us)

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 与我们联系。
